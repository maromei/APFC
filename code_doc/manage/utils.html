manage.utils.build_sim_info_str(config: dict, index: int, theta: float | None = None, add_info: str = '') → str#

Builds a latex string with additional information. The purpose is to include this string into plots to give context.

Parameters:

  • config (dict) – The config dictionary

  • index (int) – Which index of the simulation is displayed

  • theta (float | None, optional) – Angle. Defaults to None.

  • add_info (str, optional) – additional info to dislay. It will be appended at the end. Defaults to “”.

Returns:

formatted latex string

str

manage.utils.get_thetas(config: dict, use_div: bool = True, endpoint: bool = True) → array#

Creates an array of thetas based on the config. This function is supposed to supply a consistent way to get the same angles.

Parameters:

  • config (dict) – config object. Explicitely used keys are: thetaCount and thetaDiv (if use_div=True).

  • use_div (bool, optional) – Whether to divide the interval into thetaDiv parts. Where thetaDiv. Defaults to True.

  • endpoint (bool, optional) – Whether to include the last value in the range. Defaults to True.

Returns:

The equaly spaced thetas

Return type:

np.array

manage.utils.fill(arr: array, div: int, add=False) → array#

Duplicates and stacks an array div times. It is assumed that the array is cylcic and has both endpoints in there. This function is usefull if f.e. you have angle values in the range \([0, 2 \pi]\). This function will not duplicate the endpoints.

Parameters:

  • arr (np.array) – Array to duplicate

  • div (int) – How often it should be duplicated

  • add (bool, optional) – Whether the duplicated arrays should add i * max(arr) to its value. Defaults to False.

Returns:

The filled array

Return type:

np.array

manage.utils.fill_df(df: DataFrame, new_columns: list[str], div: int, add=False) → DataFrame#

Similar to manage.utils.fill(). Just that it is applied to every row in the dataframe.

Parameters:

  • df (pd.DataFrame) – original dataframe

  • new_columns (list[str]) – new columns

  • div (int) – How often it should be duplicated

  • add (bool, optional) – Whether the duplicated arrays should add i * max(arr) to its value. Defaults to False.

Returns:

filled df

Return type:

pd.DataFrame
